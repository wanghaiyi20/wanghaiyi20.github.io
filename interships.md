## China Electronics Chuntian Research Institute

**Partment:** Advanced Computing Laboratory 
**Internship Date:** 2025-12 ~ Present

Conducted technical research on RISC-V matrix extensions and AI computing acceleration. Systematically surveyed the design principles of matrix / vector instruction extensions, their integration with processor architectures, typical application scenarios, and market development trends. Focused on instruction mapping and execution flow analysis for core kernels such as GEMM and convolution, and evaluated the impact of matrix extensions on computing throughput, data movement efficiency, cache pressure, and hardware resource overhead, providing technical reference for future matrix computing processors and acceleration subsystems.

### Main Responsibilities:

- **Technical research on RISC-V matrix extensions**: Investigated RISC-V matrix / vector instruction extension schemes, and summarized instruction organization, register / Tile data structures, execution unit design principles, and hardware-software co-design characteristics.
- **Exploration of integration with processor architectures**: Studied the integration of matrix computing units with CPU cores, vector units, cache systems, on-chip interconnects, and memory subsystems. Analyzed architectural differences between integrating matrix extensions as internal CPU functional units and as coprocessor-style acceleration units.
- **Instruction mapping and execution flow analysis for typical kernels**: Analyzed data partitioning, instruction scheduling, memory access patterns, and execution flows of GEMM, convolution, and other typical kernels under matrix instructions, clarifying the acceleration path of matrix extensions for core AI workloads.
- **Performance benefit analysis of matrix extensions**: Evaluated the performance benefits and implementation costs of matrix extensions from the perspectives of computing throughput, data reuse, memory bandwidth, cache pressure, control complexity, and hardware area, providing reference for future matrix computing processor and acceleration subsystem design.

## 中国电子春天研究院

**部门:** 先进计算实验室
**实习日期:** 2025-12 ~ 至今

围绕 RISC-V 矩阵扩展与 AI 计算加速方向开展技术调研，系统梳理矩阵 / 向量类指令扩展的设计思路、处理器架构集成方式、典型应用场景及市场发展现状。重点分析 GEMM、卷积等核心算子的指令映射与执行流程，评估矩阵扩展对计算吞吐、数据搬运效率、缓存访问压力和硬件资源开销的影响，为后续矩阵计算处理器和加速子系统设计提供技术参考。

### 主要工作：

- **RISC-V 矩阵扩展技术调研**：调研 RISC-V 矩阵 / 向量类指令扩展方案，梳理指令组织方式、寄存器 / Tile 数据结构、执行单元设计思路及软硬件协同特点。
- **矩阵扩展与处理器架构结合方式探索**：调研矩阵计算单元与 CPU 核心、向量单元、缓存系统、片上互连和存储子系统之间的集成方式，分析矩阵扩展作为 CPU 内部功能单元或协处理加速单元时的架构差异。
- **典型算子的指令映射与执行流程分析**：围绕 GEMM、卷积等典型算子，分析矩阵指令下的数据划分、指令调度、访存模式和执行流程，梳理矩阵扩展对核心 AI 算子的加速路径。
- **矩阵扩展性能收益分析**：从计算吞吐、数据复用、访存带宽、缓存压力、控制复杂度和硬件面积等角度，评估矩阵扩展的性能收益与实现代价，为后续矩阵计算处理器和加速子系统设计提供参考。
